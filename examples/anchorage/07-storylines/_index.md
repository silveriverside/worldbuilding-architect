# Storylines — Index (the web)

> 剧情是网，不是线。每条线挂≥2 个共享锚点。见 `references/storylines-and-paths.md`。

## Mainline — open（明线）
**《认证》The Certificate**：跟随认证稽查员 Mira Okonkwo，从丙七召回到韦氏裁决，亲手给"谁还算本人"盖章。公开的赌注：法律如何定义一个人。
- 锚点：锚地港、港务、Mira Okonkwo。

## Mainline — hidden（暗线）
**《牧人》The Graziers**：港务并非被动扩权——它从牧带（渣带）的"捕捞"里秘密获利，并刻意压下"星尘虫可能有智慧"的证据，因为承认了，整条产业链和它的权力都要崩。**从一开始就真实存在并在制造明线事件。**
- 留在明线上的指纹：港务每次危机后扩权的"运气"太好（暗流⑤）；几份被压下的牧带声学记录（暗流④）；Mira 经手过一份"非本人"认证，对象其实是个目击了回声的猎人。
- 反转效果：当《牧人》揭开，《认证》里那些"谁算人"的争吵，会被重读为"人类连别人是不是人都不愿承认"。

## Side-arcs（支线）
- **《潮下》Undertide**：黑市作坊头面人物 Silas Quist 在召回中既是受害者也是证据持有者，他的选择决定证据流向谁——支线结局成为明线召回的触发力度。触点：潮下、丙七召回。
- **《放牧执照》The Graze Licence**：一个赏金猎人追猎"星尘虫"，逐渐听懂了"回声"——个人良心故事，却是暗线④的引信。触点：渣带、第一次回声。
