# 示例世界 · 锚地 Anchorage

> 这是用 **Worldbuilding Architect** 跑出来的一个完整 codex，用来展示方法论落地后长什么样。
> 它不是教程，是**真实产物**——你可以直接当成一个可续写的世界来读、来扩展。

## 一句话

人类把第一座太空电梯锚在赤道潮滩上，于是有了"锚地"。三代人之间，争论从"谁拥有你义肢里的固件"滑向"被恢复的备份还算不算你"，最后撞上一个谁也没料到的事实：那条被当成垃圾的轨道尘带，是**别人的海**。

**核心问题**：当意识能被留存、身体能被替换时，"延续"意味着什么——以及谁有资格延续？

## 它展示了什么

- 一条会"变异"的时代主轴：从「系链时代」（谁拥有我义肢的固件）→「退线纪」（被恢复的备份算不算人）→「双生纪」→「退潮纪」（"我们"还只指人类吗）。每个时代争论的**问题种类**都变了，而不只是科技升级。
- 两个**镜像派系**搁架众 / 落地党，同生于"韦氏裁决"——同一场判决的正反两面，各自内部还拧着三股不相容的暗劲。
- 一次完整的**质变**演示（丙七召回），从一个稽查员的例行签字一路级联到立法危机。
- 三条**世界线**（本线 α / 干线 β1 / 静线 β2）全部以"对主干的 diff"撰写，最终都向同一个不动点「第一次回声」收束，并有跨线闭环钩子（β2 崩溃时"留一道回声"，被 α 的主角当作既视感收到）。
